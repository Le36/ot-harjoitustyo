# OT-harjoitystyö #

**CHIP-8** *Emulator*

## Komentorivitoiminnot ##

### Sovelluksen käynnistäminen
Sovelluksen voi käynnistää komennolla

```
mvn compile exec:java -Dexec.mainClass=com.chip8.emulator.Main
```

### Testaus
Testit suoritetaan komennolla
```
mvn test
```
